# Crypto Lab (work in progress)

**Crypto Lab** — это исследовательская лаборатория для крипторынка на Python.

Цель проекта — дать удобную и воспроизводимую среду для:

- загрузки и нормализации исторических OHLCV-данных;
- проверки качества данных (QC) по жёсткому data-contract;
- быстрой проверки торговых гипотез через простой бэктестер;
- анализа результатов в ноутбуках и отчётах.

## Статус

Work in progress.  
Сейчас в фокусе:

- определение data-contract для свечей;
- разработка загрузчика данных через `ccxt`;
- базовый модуль QC для проверки целостности данных.

## Структура (планируемая)

- `docs/` — vision, data-contract, QC-правила, roadmap.
- `lab_core/` — ядро лаборатории:
  - `loader/` — загрузчики данных;
  - `qc/` — проверка качества данных;
  - `backtester/` — бэктест-движок;
  - `strategies/` — простые стратегии.
- `research/` — Jupyter-ноутбуки и шаблоны исследований.
- `tests/` — pytest-тесты.

## Использование

Инструкции по установке и примеры использования появятся после реализации первых версий:

- загрузчика данных (`loader v0`);
- базового QC;
- минимального бэктестера.
